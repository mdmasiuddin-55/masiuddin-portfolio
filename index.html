Pace Healthcare App

Developed and deployed a scalable and secure cloud infrastructure for a comprehensive healthcare application, ensuring high availability and fault tolerance. Leveraged AWS services such as EC2, RDS (MySQL), S3, VPC, Elastic Load Balancer (ELB), and CloudWatch for monitoring and performance tracking. Used Terraform to implement Infrastructure as Code (IaC), allowing consistent and repeatable environment provisioning. Configured Route 53 for DNS management and implemented IAM roles and security groups to enforce strict access control. The project significantly improved deployment efficiency, operational consistency, and the overall security posture of the healthcare system.
